# Install RAKI Verbalizer

## build Docker Endpoint for DRILL and name the image `drill`
https://github.com/dice-group/RAKI-Drill-Endpoint

## build Docker Endpoint for Verbalizer and name the image `kaki-verbalizer-webapp`

Run the script:
```bash
		build.sh
```


# Run RAKI Verbalizer

Run the script:
```bash
		docker-compose up
```
